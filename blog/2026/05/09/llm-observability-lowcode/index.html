一个大模型应用从 Demo 到生产，中间隔着一整套工程化体系：

一、LLM 可观测性

1.1 为什么 LLM 调用需要追踪？

传统 API 调用：request → response，确定性输入输出。
LLM 调用：prompt → token序列 → 概率采样 → response，非确定性。

需要追踪的不只是”调了没有”，而是：

  • Prompt 长了多少 Token？
  • 模型返回了什么？
  • 耗时多少？花了多少钱？
  • 哪一步出错的？

1.2 Langfuse（推荐自部署）

Langfuse 是开源的 LLM 可观测性平台，支持自部署。

1.3 LangSmith

LangSmith 是 LangChain 团队的商业产品，与 LangChain 生态深度集成。
