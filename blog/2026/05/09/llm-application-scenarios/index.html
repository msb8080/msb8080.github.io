## 八、场景选型总结

| 场景 | 核心技术 | 实现难度 | ROI |
|------|----------|----------|-----|
| 智能客服 | RAG + Function Calling | ⭐⭐⭐ | 很高 |
| 代码助手 | 长上下文 + Diff 分析 | ⭐⭐⭐ | 高 |
| Text-to-SQL | Schema 理解 + SQL 生成 | ⭐⭐⭐⭐ | 高 |
| 内容生成 | Prompt + RAG | ⭐⭐ | 中 |
| 自动化工作流 | Agent + 编排 | ⭐⭐⭐⭐ | 高 |
